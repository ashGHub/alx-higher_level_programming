The ``MyList`` module
========================

Using ``print_sorted``
-----------------------

First import `MyList`` from ``MyList`` class

	>>> MyList = __import__('1-my_list').MyList

Now use it:

	>>> a = MyList()
	>>> a.print_sorted()
	[]
	>>> a.append(1)
	>>> a.print_sorted()
	[1]
	>>> a.append(4)
	>>> a.append(2)
	>>> a.print_sorted()
	[1, 2, 4]
